# Lineage

This is the manual easter egg.

Orphan Radar is a wink to PageRank and information theory, but it is not a PageRank clone.

- **PageRank lineage:** graph authority is useful, but global popularity is not the same as local relevance.
- **Information-theory lineage:** a good link reduces uncertainty. `[[AI]]` often reduces little; `[[Agent Reliability]]` may reduce more.
- **Curation lineage:** the system proposes; the human reviewer decides.

Small motto:

```text
Hubs route attention. Specific links reduce uncertainty. Review keeps the graph honest.
```
